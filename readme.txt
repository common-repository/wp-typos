Plugin adds text with typos in post ending to get more hits from search engines.

== Description ==

Plugin is usefull for Russian webmasters. It adds post title to the end of the post with another keyboard layout.For example compare google search results <a href="http://www.google.com.ua/search?q=Left+4+Dead+2">Left 4 Dead 2</a> and <a href="http://www.google.com.ua/search?q=%E4%F3%E0%E5+4+%E2%F3%F4%E2+2">‰Û‡Â 4 ‚ÛÙ‚ 2</a>. 

== Installation ==

1. Upload '/wp-typos/' to the '/wp-content/plugins/' directory2. Activate the plugin through the 'Plugins' menu in WordPress
3. This plugin automaticly adds typos to the end of single post.
